/转\\
使用QDir::toNativeSeparators
```c++
QString path = "C:/temp/test.txt";
convertpath = QDir::toNativeSeparators(path);
```
输出
```C:\\temp\\test.txt```

\转/
使用QDir::fromNativeSeparators
```c++
QString path = "C:\\temp\\test.txt";
convertpath = QDir::fromNativeSeparators(path);
```
输出
```C:/temp/test.txt```
